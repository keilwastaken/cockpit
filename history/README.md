# Historical Evidence

These scorecards document retired Cockpit routing and worker experiments. They are not evidence for current Cockpit Lite behavior.

The experiments showed that strict routing could reduce expensive-model usage for broad research, but generally increased total tokens. Supervised mutable delegation added substantial parent/worker duplication and did not produce a reliable economic win. The recorded commits and benchmark-source hashes in each scorecard identify the exact implementation used; Git history preserves those sources.

Current guidance is intentionally smaller: native OpenCode owns routing and execution, built-in `explore` may use a cheaper model for cold broad research, and an optional `hands` subagent is reserved for substantial mechanical low-risk work.

No general token- or cost-saving claim is supported.
